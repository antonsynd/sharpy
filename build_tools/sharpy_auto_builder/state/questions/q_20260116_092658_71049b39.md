## Decision: Fully-Qualified Names with Exports Class Pattern

Each Sharpy module compiles to a static class with a nested `Exports` class:

```python
# config.spy
MAX_SIZE: int = 100
```

```csharp
// Config.cs
namespace MyProject
{
    public static class Config
    {
        public static class Exports
        {
            public static int MaxSize = 100;
        }
    }
}
```

**Import translation:**
- `import config` → `using config = MyProject.Config.Exports;`
- `from config import MAX_SIZE` → `using static MyProject.Config.Exports;`

**Member access:**
- `config.MAX_SIZE` → `config.MaxSize` (via using alias + name mangling)
