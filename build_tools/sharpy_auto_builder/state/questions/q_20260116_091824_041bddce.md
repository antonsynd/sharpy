**Pattern**: `EnumType.MEMBER_NAME`

**Transformation**:
- Sharpy: `Color.RED`

**Implementation**:

- If symbol is a `TypeSymbol` with `TypeKind.Enum`, applies special handling
- Enum type name: `NameMangler.ToPascalCase(enumTypeIdentifier.Name)`

#### 2. **Enum Value Property** (lines 3040-3051)
**Pattern**: `enum_instance.value`

**Transformation**:
- Sharpy: `color.value` (where `color` is an enum instance)
- C#: `(int)color`

**Implementation**:
- Checks if member name equals "value" (case-insensitive)
- Uses `IsEnumTypeExpression()` to verify the object is an enum type
- Generates a cast to `int` instead of member access

**Note**: There's a potential bug here - `IsEnumTypeExpression()` only handles `Identifier` expressions, not `MemberAccess` expressions like `Color.GREEN.value`.
