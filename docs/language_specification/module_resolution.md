# Module Resolution

## Name Conversion

- Module names converted: `snake_case` → `PascalCase`
- Common acronyms uppercased: `io`, `ui`, `xml`, `html`, `api`, `sql`, `db`, `http`, `ftp`, `smtp`, `tcp`, `udp`, `ip`, `uri`, `url`, `json`, `csv`, `guid`
- Example: `system.collections.generic` → `System.Collections.Generic`
